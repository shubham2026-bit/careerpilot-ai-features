ANSWER: What environment variables do you need to add?

REQUIRED (3 Main Variables):
───────────────────────────────────────────────────────────────────────────────

1. DATABASE_URL
   Where: Neon PostgreSQL
   Format: postgresql://user:password@host:port/database?sslmode=require
   How to get:
     → Go to neon.tech
     → Create account & database
     → Copy connection string
     → Make sure to include your password!
   
   Example:
   DATABASE_URL=postgresql://careerpilot_user:MyPassword123@ep-calm-lake-1234.us-east-1.neon.tech/careerpilot?sslmode=require

2. BETTER_AUTH_SECRET
   Where: Generate yourself
   Format: Random 32-byte base64 string
   How to get:
     → Run: openssl rand -base64 32
     → Copy the output
   
   Example:
   BETTER_AUTH_SECRET=aBcDeFgHiJkLmNoPqRsTuVwXyZ1234567890abc=

3. RESEND_API_KEY
   Where: Resend Dashboard
   Format: re_xxxxxxxxxxxxx
   How to get:
     → Go to resend.com
     → Create account (free)
     → Go to API Keys
     → Copy key
   
   Example:
   RESEND_API_KEY=re_abc123def456ghi789jkl

OPTIONAL (Not needed - Auto-configured):
───────────────────────────────────────────────────────────────────────────────

BETTER_AUTH_URL
   Status: Auto-configured by Vercel
   Local: http://localhost:3000
   Production: Auto-set to your domain

AI_GATEWAY_API_KEY
   Status: Not needed! Uses free Vercel AI Gateway
   Only use if: You want a specific AI provider (xAI, Deep Infra, etc.)

═══════════════════════════════════════════════════════════════════════════════

WHERE TO ADD THEM (Vercel):
───────────────────────────────────────────────────────────────────────────────

1. Go to vercel.com
2. Select your project
3. Click "Settings" (top right)
4. Click "Environment Variables" (left sidebar)
5. Add each variable:
   - Key: DATABASE_URL
   - Value: [copy from Neon]
   - Click "Save"
   
   Repeat for:
   - BETTER_AUTH_SECRET
   - RESEND_API_KEY

6. Click "Deploy" or push to GitHub (auto-deploys)

LOCAL DEVELOPMENT (.env.local):
───────────────────────────────────────────────────────────────────────────────

Create file: .env.local

Add these 3 lines:
DATABASE_URL=postgresql://user:password@localhost:5432/careerpilot
BETTER_AUTH_SECRET=your_generated_secret_here
RESEND_API_KEY=re_your_api_key_here

Then run: pnpm dev
